**Simplificação do Processo de Registro de Captura**
- **Objetivo**: Digitalizar o processo de registro de captura para torná-lo mais eficiente e menos demorado para os pescadores.

**Transparência e Rastreabilidade**
- **Meta**: Fornecer maior transparência na cadeia de suprimentos, permitindo melhor rastreabilidade da captura do mar até o consumidor.

**Sustentabilidade**
- **Foco**: Garantir que o processo de registro de captura atenda aos requisitos de sustentabilidade, promovendo práticas de pesca sustentáveis.

**Otimização de Processos Internos**
- **Benefício**: Digitalizar o processo de registro de captura para otimizar processos internos, tornando-os mais eficientes e econômicos.

Esta chamada destaca os principais objetivos e benefícios do projeto de transformação digital da Royal Greenland, utilizando a Plataforma de Tecnologia de Negócios SAP (SAP BTP) para melhorar a eficiência, transparência e sustentabilidade nas operações de pesca.
